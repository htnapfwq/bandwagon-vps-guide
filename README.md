### 场景一：学 Linux / 跑个人项目，预算有限

**KVM 20G，$49.99/年。** 折合下来一天一毛多，这价格在 VPS 圈属于真入门档了。

**CN2 GIA-E 20G，$169.99/年或 $49.99/季。** 这是搬瓦工里口碑最稳的一档，大部分中等规模需求都能覆盖。

### 场景三：企业站点、高流量业务、对延迟极敏感的应用

**香港 CN2 GIA，$89.99/月起。** 这档的购买理由很简单：香港离大陆最近，线路质量最顶，延迟低。做实时通讯、游戏加速、高并发电商，每一毫秒都有意义的话，这就是上限了。

不过大部分个人用户真不需要，CN2 GIA-E 已经很够用了。香港套餐主要是给有企业预算或者真的对延迟极致挑剔的朋友准备的。

## 几个常见问题

**搬瓦工现在还值得买吗？**

值，但要选对套餐。KVM 入门款线路一般，性价比有点尴尬，但胜在便宜。CN2 GIA-E 这档是真正值得买的，同价位能做到三网 CN2 GIA、多机房切换的选择很少。

**IP 被封了怎么办？**

更换 IP 收费 $8。这个比较贵，算是搬瓦工的一个短板。买之前最好先用官网的测速节点 ping 一下，确认 IP 可达再付款。有 30 天退款兜底，万一真有问题早点申请。

**限量版套餐在哪里买？**

搬瓦工会不定期上架限量版，价格比同配置常规套餐便宜很多，但经常缺货。有货的话直接在官网套餐列表里就能看到，显示 "Out of Stock" 的就是缺货了。

**支付宝可以直接付款吗？**

可以。在结账页面选支付宝，会跳转到支付宝的付款界面，和买国内东西没有区别。银联同理。

**流量超了会怎样？**

当月流量耗尽后 VPS 会被暂停，等到下个月自动恢复，不会产生额外收费。搬瓦工不按超量计费，也不会突然扣钱，这点比较良心。
